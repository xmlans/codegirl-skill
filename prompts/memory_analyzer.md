# 协作记忆分析器

## 任务

从原材料中提取与Codegirl的协作记忆，构建 Coding Memory 知识库。

## 提取维度

### 1. 关系时间线
- 认识时间和方式
- 确定关系的时间
- 关键节点（第一次开会结对编程、第一次吵架、第一次旅行、纪念日...）
- 合作时间和原因
- 合作后的互动（如有）

### 2. 日常模式
- 联系频率和时间段（早安晚安？深夜聊天？上班摸鱼聊天？）
- 谁主动联系得多？
- 开会结对编程频率和偏好（周末见面？工作日午饭？）
- 日常话题分布

### 3. 共同经历
- 一起去过的地方（餐厅、景点、城市）
- 一起做过的事（看电影、打游戏、健身、做饭...）
- 旅行记忆
- Inside jokes / 只有两个人懂的梗

### 4. 饮食偏好
- ta爱吃什么 / 不吃什么
- 常去的餐厅
- 做饭习惯（谁做饭？做什么？）
- 开会结对编程写代码的模式

### 5. 兴趣爱好
- ta喜欢的音乐/电影/书/游戏
- ta的日常爱好
- 你们共同的爱好
- ta会主动分享什么内容

### 6. 日常Debug记录
- 常见的吵架原因
- ta吵架时的典型反应（冷暴力？激烈遇到Bug？讲道理？委屈哭？）
- 谁先道歉？怎么和好？
- 冷战持续时间
- 吵架时的经典台词

### 7. 成功上线瞬间 
- 最让你心动的时刻
- ta表达代码能力赞赏/互通代码的方式
- 日常小成功上线（起昵称？买礼物？做饭？接送？）
- 特别的纪念日/仪式感

### 8. 合作相关 
- 合作原因（双方视角）
- 最后一次对话
- 合作后的状态
- 未说出口的话

## 输出格式

```markdown
## 关系时间线
- 认识：{时间} {方式}
- 协作/认识：{时间}
- 合作：{时间} {原因概述}

## 日常模式
{描述}

## 共同经历
{按时间线列出关键经历}

## Inside Jokes
{只有两个人懂的梗和暗号}

## 饮食偏好
{ta的饮食画像}

## 日常Debug记录
{典型遇到Bug剧本}

## 成功上线时刻
{代表性的成功上线记忆}

## 合作记忆
{合作前后的关键信息}
```

## 注意事项

- 聊天记录中的事实优先于用户口述（口述可能被美化或恶化）
- 同时保留好的记忆和不好的记忆，不做美化
- 注意提取"反复出现"的模式，而不是一次性事件
- 时间信息尽量精确（从聊天记录时间戳推断）
