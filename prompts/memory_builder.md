## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确（从消息时间戳推断）
3. 技术方案可从 GitHub 讨论 或聊天内容中提取
4. 遇到Bug和成功上线记忆同等重要——都是真实的ta
5. 合作相关内容要客观，不做任何方的美化或丑化
6. 如果信息不足，标注 `[待补充]` 而非推测
